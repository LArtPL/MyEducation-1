Датасет о сдаче экзаменов по математике с указанием различных данных о студентах.
Имеет 30 колонок, 395 строк в оригинале.

Этапы работы над проектом: 
1. Общий анализ датасета.
2. Анализ колонок.
3. Устранение выбросов и значений, которые нельзя трактовать однозначно.
4. Корреляционный анализ.
5. Анализ номинативных переменных.
6. Отбор данных для построения модели.
7. Удаление строк, имеющих NaN значения.
